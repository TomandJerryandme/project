本文主要是记录与解释shell命令中的操作符的使用
众所周知，在计算机语言中，基本操作符分为：算术运算符、关系运算符、逻辑运算符、布尔运算符等
在高级语言中，还存在位运算符、成员运算符、身份运算符等。
在shell中，还有文件测试运算符与字符串运算符

注意：在shell中，基本上不需要空格来分隔，因为可能会报语法错误（除了判断语句[ 条件表达式 ]）
在判断语句[]中，各个变量与操作符之间要用空格分开，而且要与左右中括号分开

一、算术运算符
    在shell中，算术运算符与其他类似，都是+ - * / % = == ！= 这几种，但是没有MOD。
    其中，== 与 != 的使用界限比较模糊，可以在test这种条件语句中使用，但是不能在[] 这种条件语句中使用

二、关系运算符
    关系运算符只支持数字，不支持字符串，除非字符串的值是数字
    关系运算符： -eq 等于
                -ne 不等于
                -gt 大于
                -lt 小于
                -ge 大于等于
                -le 小于等于

三、布尔运算符
    布尔运算符只能运算布尔值（不要与高级语言的位运算混淆）
    !   非运算，
    -o  或运算
    -a  与运算

四、逻辑运算符
    一般用于多个表达式之间的连接
    eg: a == False  b = True
    a && b ---->   a AND b
    a || b ---->   a OR b
    &&  AND
    ||  OR
    [] 与[[ ]] 的区别：
        [[ ]] 中逻辑组合可以使用 &&  || 符号
        而[] 里面逻辑组合可以用  -a   -o

五、字符串运算符
    eg: a == 'abc' b == 'efg'
    =   : eg [ $a = $b ] ----> 返回false
    !=  : [ $a != $b ] ----> true
    -z  : [ -z $a ] ----> false  判断a长度是否为0，为0 则返回true
    -n  : [ -n $a ] ----> true   判断a长度是否为0，不为0返回true
    字符串本身   ：  [ $a ]  检验 a 是否为空、类似高级语言

六、文件测试运算符
    操作符                      说明                                                                                 举例

    -b file       检测文件是否是块设备文件，如果是，则返回 true。                                                    [ -b $file ] 返回 false。

    -c file       检测文件是否是字符设备文件，如果是，则返回 true。                                                  [ -c $file ] 返回 false。

    -d file       检测文件是否是目录，如果是，则返回 true。                                                         [ -d $file ] 返回 false。

    -f file       检测文件是否是普通文件（既不是目录，也不是设备文件）,如果是，则返回 true。                           [ -f $file ] 返回 true。

    -g file      检测文件是否设置了 SGID 位，如果是，则返回 true。                                                  [ -g $file ] 返回 false。

    -k file       检测文件是否设置了粘着位(Sticky Bit)，如果是，则返回 true。                                       [ -k $file ] 返回 false。

    -p file       检测文件是否是具名管道，如果是，则返回 true。                                                     [ -p $file ] 返回 false。

    -u file       检测文件是否设置了 SUID 位，如果是，则返回 true。                                                 [ -u $file ] 返回 false。

    -r file         检测文件是否可读，如果是，则返回 true。                                                         [ -r $file ] 返回 true。

    -w file       检测文件是否可写，如果是，则返回 true。                                                           [ -w $file ] 返回 true。

    -x file        检测文件是否可执行，如果是，则返回 true。                                                        [ -x $file ] 返回 true。

    -s file        检测文件是否为空（文件大小是否大于0），不为空返回 true。                                          [ -s $file ] 返回 true。

    -e file        检测文件（包括目录）是否存在，如果是，则返回 true。                                               [ -e $file ] 返回 true。
